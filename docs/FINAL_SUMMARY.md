## User Journey

### Before (Traditional Cards)
1. Click "Draw Card" button
2. Generic card flip animation
3. See suit and value
4. Click continue

**Emotional Response**: Boring, generic, forgettable

### After (Neural Interface)
1. Click **"INTERCEPT FRAGMENT"**
   - Grid accelerates, particles rush
   - Button changes to "INTERCEPTING..."

2. **Fragment materializes**
   - Appears with 3D rotation
   - Glitches with RGB split (3 cycles)
   - Bio-pulse rings expand outward
   - Cyan border ignites with dual neon glow

3. **Text reveals**
   - Narrative materializes character-by-character
   - Fragment ID displays in cyan monospace
   - Corruption scanner sweeps across surface

4. Click **"CONTINUE"**
   - Fragment floats upward
   - Brightens and blurs
   - Dissolves into data stream

**Emotional Response**: Engaging, thematic, memorable - **signature moment**
